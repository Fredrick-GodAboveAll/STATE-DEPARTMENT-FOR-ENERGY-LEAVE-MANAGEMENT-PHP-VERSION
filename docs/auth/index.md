# Authentication Documentation

This folder contains the authentication documentation for the PHP Leave Management System.

## Files

- `AUTHENTICATION_DOC.md` — Overview of authentication flow, CSRF protection, password reset, session, and logout.
- `authentication-system-guide.md` — Full implementation guide for building and understanding the auth module.
- `SECURITY_IMPROVEMENTS.md` — Notes on security improvements such as lockout, session timeout, and email verification.
- `EMAIL_SETUP.md` — Email setup notes for password reset and SMTP configuration.

## How to use this folder

1. Read `AUTHENTICATION_DOC.md` first for the basic auth flow.
2. Use `authentication-system-guide.md` for full file-by-file implementation details.
3. Refer to `SECURITY_IMPROVEMENTS.md` for the security enhancements you applied.
4. Use `EMAIL_SETUP.md` when configuring password reset email delivery.
